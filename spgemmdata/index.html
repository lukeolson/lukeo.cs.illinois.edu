Data for SpGEMM:

This contains all matrices for the SpGEMM tests presented in Optimizing Sparse Matrix-Matrix Multiplication for the GPU, by Steven Dalton, Nathan Bell, and Luke N. Olson. Each A matrix has an companion matrix P in the directory. The storage scheme appends "_P" to the end of the A matrix filename.

The full set of matrices are found here: https://zenodo.org/record/7763972
